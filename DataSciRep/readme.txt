org.Hs.core_paths.Rdat contains

corum_core - named list of genes constituting CORUM-annotated human protein complexes.
Data from http://mips.helmholtz-muenchen.de/corum/download/coreComplexes.txt.zip 

kegg_path - named list of genes constituting KEGG-annotated human pathways (https://www.genome.jp/kegg/)

org.Hs - data.frame containing:
ENSG - Ensembl gene id
EntrezID - Entrez gene id
symbol - Gene symbol

trxL - data.frame containing:
length - mRNA lengh, bp
gc - GC content
